## Restaurant MDI
# Ryan Crawford
### Description:

This is an MDI (multiple document interface) for a restaurant that manages orders for tables
and calculates the totals for each table, then prints out a ticket.

### Input
The user loads a menu into the application with product names and prices, along with the
category for each (beverage, appetizer, entree, or dessert).

### Output
A ticket is printed out showing the orders for each table along with totals.

|   #   | File                       | Description                                                |
| :---: | -------------------------- | ---------------------------------------------------------- |
|   1   | [CustomerForm.cs](./CustomerForm.cs)     | Customer form used to submit orders for a table                                            |

|   4   | [TableForm.cs](./TableForm.cs)   | Table form used to list items and prices for a table                                      |
